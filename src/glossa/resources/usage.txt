Τρόπος χρήσης:
    java -jar glossa-interpreter.jar παράμετροι τοποθεσία_αρχείου_πηγαίου_κώδικα
π.χ.
    java -jar glossa-interpreter.jar /home/user/Documents/Hello_World.gls

Παράμετροι
    Όλες οι παράμετροι είναι προαιρετικές.
    -h  : εμφανίζει αυτό το μήνυμα
    -V  : εμφανίζει την έκδοση του διερμηνευτή
    -i  : εκτέλεση βήμα-προς-βήμα
    -f τοποθεσία_αρχείου_εισόδου
        : Η είσοδος των εντολών ΔΙΑΒΑΣΕ
          γίνεται από το αρχείο εισόδου,
          διαβάζοντας μία γραμμή ανά εντολή ΔΙΑΒΑΣΕ
    -g  : εκτέλεση βήμα-προς-βήμα μέσα από Περιβάλλον Γραφικής Διεπαφής Χρήστη
